# frontend-dev-test
Frontend developer competency test. This will be a function single-page app where the user will retrieve all articles and display in table format.

## To build the project
run the following commands:
```
npm install
npm install -g bower
bower install
```

## To run
run the following commands:
```
gulp
node app.js
```
